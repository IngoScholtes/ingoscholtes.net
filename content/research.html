Insights

  1. pathpy Tutorial

    This educational tutorial introduces the analysis of data on pathways and temporal networks using the python package pathpy. It is based on the multi-order graphical modeling framework introduced in my recent work.

  2. Media coverage: Temporal Network Analytics

    A summary of my research on new data mining techniques for time-stamped and sequential data on complex networks can be found in this ETH News article.

  3. Temporal Networks Tutorial

    This educational python tutorial illustrates (and reproduces) our recent works about the use of higher-order network topologies to analyze temporal networks with non-Markovian characteristics.

  4. pyTempNets

    The python module pyTempNet can be used to study temporal networks based on higher-order network models. This module is deprecated and superseded by pathpy.

  5. Self-Portrayal of Research

    In the article Understanding Complex Systems: When Big Data meets Network Science I discuss applications of relational data mining in the context of Big Data.

  6. Media coverage: Predicting Scientific Success

    This article in EuroScientist reports on our work, showing that the citation success of scientists can be predicted, with surprising accuracy, based on their centrality in the coauthorship network.
